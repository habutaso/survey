## 3. 補足：複数ユニットにまたがるストーリーの扱い
- **US-207（オフライン入力＋提出時同期）**: クライアント挙動は U6f が主。サーバ受け口 `ingestSubmission` は U2。暗号化は U-Cross。
- **US-301〜303 / US-602〜603（入力系）**: 入力データ実体は U2（first/secondSurvey）、計算反映は U3a/U3b、入力 UI は U6u。主は計算ユニットに置き、データ・UI を関与で明示。
- **US-404（階按分）**: 共有関数 `applyFloorRatio` は U3c が主。第1次/第2次双方が利用、UI は U6u。
- **US-806**: デモ削除は U0、セキュアな既定動作（CSP/エラー秘匿/fail closed）は U-Cross が担う。
